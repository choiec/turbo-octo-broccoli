# Security

## Supported versions

We release security updates for the current stable branch.
Check the
[releases](https://github.com/cherry-pick/picker/releases)
page for supported versions.

## Reporting a vulnerability

If you believe you have found a security vulnerability,
please report it to
**[security@cherry-pick.net](mailto:security@cherry-pick.net)**.
Do not open a public issue.

- **What to include**: Description of the issue, steps to
  reproduce (if possible), and impact. You may report
  anonymously; we will not require your identity.
- **What to expect**: We will acknowledge receipt and aim to
  respond with next steps within a reasonable time. We may
  ask for more details and will keep you updated on the
  status of the fix.
- **Disclosure**: We ask that you allow us a reasonable time
  to address the issue before any public disclosure. We will
  credit you in advisories if you wish.

This project is maintained by
[cherry-pick](https://cherry-pick.net).
